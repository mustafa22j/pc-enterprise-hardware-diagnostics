# PC Enterprise Hardware Diagnostics

## Overview
This repository contains structured documentation from hands-on PC troubleshooting and hardware service labs.

It demonstrates technician-level workflows used in real-world IT support and enterprise desktop environments.

---

## Included Case Studies

### BIOS / POST Troubleshooting
Structured workflow for diagnosing:
- No POST
- Boot loops
- Incorrect boot order
- Storage detection failures
- UEFI/Legacy configuration issues

### RAID Configuration & Validation
- RAID concepts (0, 1, 5, 10)
- Firmware RAID setup
- Disk initialization
- Validation and rebuild testing
- Troubleshooting RAID mode mismatches

### Windows Forced Logoff / Shutdown Case
- Software-triggered shutdown analysis
- `shutdown -a` mitigation
- Startup folder inspection
- Task Scheduler investigation
- Registry Run key validation
- Verification testing

### Desktop Teardown & Rebuild Procedure
- Safe disassembly workflow
- ESD handling
- Cable management
- Power connector validation
- BIOS verification after rebuild

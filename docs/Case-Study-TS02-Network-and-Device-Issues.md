\# Case Study: TS-02 – Network and Device Issues



\## Overview

Diagnosed and resolved multiple hardware and software issues affecting system functionality.



\---



\## Issues Identified

\- No network connectivity (NIC disabled in BIOS)

\- Battery not functioning

\- Touchpad not working

\- No audio output

\- Wi-Fi not working (service disabled)

\- Ethernet blocked by firewall



\---



\## Troubleshooting Steps



\### 1. Network Issue (NIC)

\- Entered BIOS

\- Enabled onboard NIC



\### 2. Battery Issue

\- Inspected internal connections

\- Determined battery likely faulty



\### 3. Touchpad Issue

\- Adjusted BIOS settings to PS/2 mode



\### 4. Audio Issue

\- Updated drivers via Device Manager



\### 5. Wi-Fi Issue

\- Enabled WLAN AutoConfig service



\### 6. Ethernet Issue

\- Reset firewall:

netsh advfirewall reset



\---



\## Result

\- Full system functionality restored

\- Estimated repair time: \~90 minutes

\- Driver troubleshooting

\- Network diagnostics

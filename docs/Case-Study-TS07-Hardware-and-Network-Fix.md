\# Case Study: TS-07 – Hardware and Network Repair



\## Overview

Diagnosed multiple system-level issues including hardware misconfiguration and network failures.



\---



\## Issues Identified

\- Power button not working

\- GPU not displaying

\- No network connectivity (APIPA address)

\- DHCP service disabled

\- Missing chipset drivers



\---



\## Troubleshooting Steps



\### 1. Power Issue

\- Fixed motherboard jumper configuration



\### 2. GPU Issue

\- Set primary display to PCIe in BIOS



\### 3. Network Issue

\- Enabled DHCP Client service



\### 4. Driver Issue

\- Identified missing chipset drivers



\---



\## Result

\- System fully operational

\- No hardware replacement required



\---



\## Skills Demonstrated

\- Hardware diagnostics

\- BIOS configuration

\- Windows networking

\- Driver troubleshooting
